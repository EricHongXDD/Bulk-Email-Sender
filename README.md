# Bulk-Email-Sender
 Batch sending emails to individual recipients with GUI interface.批量发送单收件人邮件，带有GUI界面。
 
 # Usage
 mail_host：SMTP服务器地址

 port：SMTP服务器端口

 mail_user：用户名

 mail_pass：口令密码

 sender：发件人邮箱
